Trong quá trình phát triển phần mềm, ngoài các kĩ năng về nền tảng, ngôn ngữ đang sử dụng thì việc làm chủ được **IDE** (integrated development environment) sẽ giúp cho quá trình coding được nhanh chóng và hiệu quả hơn. Sau đây mình xin giới thiệu với các bạn một số **shortcuts** trong **Xcode** - IDE để phát triển các ứng dụng iOS.

### 1. Left - Right - Bottom Menu:

- Show/Hide Debug Area (Bottom): `Shift + Command + Y`
- Show/Hide Navigator (Left): `Command + 0`
- Show/Hide Utilities (Right): `Option + Command + 0`

![alt](https://images.viblo.asia/6817d52f-56db-40c7-a081-ef51eb5a0255.gif)
### 2. Navigator & Utilities:
Để di chuyển giữa các tabs trong:
- Utilities (Right Menu): `Command Option 1–6`
- Navigator (Left Menu): `Command 1–9`

![alt](https://images.viblo.asia/2a224a3a-b0b2-4796-8adc-de50195170b6.gif)
### 3. Quick Open:
Để mở một class, func hay file... nào đó
- `Command Shift o`

![alt](https://images.viblo.asia/452961b3-d031-4b1a-ba8b-2dfb6cec6d64.gif)
### 4. Open Line Number:
Để di chuyển đến line number trong source file đó.
- `Command L`

![alt](https://images.viblo.asia/f6cf993a-d643-494d-b9da-167e17f2ae00.gif)
### 5. Assistance Editor
- Show: `Option Command Enter`
- Hide: `Command Enter`

![](https://images.viblo.asia/bfda8cf5-8710-49e3-a806-ecd046eb1131.gif)
### 6. Indent
- `Control + I`

![](https://images.viblo.asia/c6b67b5d-1247-42cf-9c14-af6de24a1a00.gif)
### 7. Hide Code:
- Hide: `Command Option Left`
- Open: `Command Option Right`

![](https://images.viblo.asia/37442469-866a-4fb1-b261-b64be6e484ee.gif)
### 8. AutoCompletion:
- `ESC` or `Command + Space`

![](https://images.viblo.asia/a947cf90-cf25-45d3-b147-67b931415f81.gif)
### 9. Move Line:
- Left: `Command + [`
- Right: `Command + ]`
- Up: `Command + Option + [`
- Down: `Command + Option + ]`

![](https://images.viblo.asia/f371f05a-acd9-4a07-9f73-a4ad65b8f19d.gif)
