Trong VSCode, muốn zoom in code lên cho to dễ nhìn thì nó lại zoom nguyên cả cái ứng dụng. Cũng hay thật nhưng đôi khi chúng ta chỉ mún phóng to code thôi thì làm như nào?

**Cách 1**: là vào chỉnh thủ công fontSize to lên (hơi phiền)

**Cách 2**: có extension hỗ trợ luôn, tên như hình bên dưới:

![](https://images.viblo.asia/2713503c-bc79-4b7d-8768-6213ba364c2d.jpg)

Và phím tắt để xài là: 
* Increase font size: **Ctrl/Cmd** and **+**
* Decrease font size: **Ctrl/Cmd** and **-**
* Reset font size to default: **Ctrl/Cmd** and **0**

Lưu ý là dấu + và - kế bên phím Back Space á nha, chứ xài bên hàng phím số thì mặc định sẽ zoom nguyên cái app.
