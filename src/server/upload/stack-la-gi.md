Đầu tiên chúng ta sẽ nói qua ngắn gọn về khái niệm : stack là tập hợp các phần mềm, các công nghệ đi kèm,những thứ này gọi chung là nền tảng để 1 ứng dụng có thể hoạt động được. 
Cấu tạo của Stack ra sao?
Một stack thường được cấu tạo bởi các thành phần:

Hệ điều hành
Web Server
Database Server
Back-end 
Programming Language

Giả sử với LAMP Stack, các thành phần này sẽ lần lượt là:

Linux
Apache
MySQL hoặc  MariaDB
PHP hoặc Python
Nên chọn stack nào ?
Tùy theo khả năng và level của người dùng mà sẽ chọn cho mình 1 stack thích hợp .
