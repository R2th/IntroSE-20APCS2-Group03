![](https://images.viblo.asia/e9862d14-d3e1-47cb-8871-8ab1c2cabada.jpeg)

Theo khảo sát Stack Overflow, React đã trở nên phổ biến gần 150% từ năm 2017 đến năm 2018. Điều này không thực sự đáng ngạc nhiên, và trong số những thứ khác đang xảy ra nhờ hệ sinh thái dựa trên component-based ngày càng phát triển của nó.


Sau đây là các thư viện UI components dành riêng cho ReactJs.

### 1. [Material-UI](http://www.material-ui.com/#/)
Là tập hợp các components của React được thiết kế theo chuẩn Material Design của Google. Với hơn [38k Stars](https://github.com/mui-org/material-ui), nó là một thư viện React UI được sử dụng phổ biến và rộng rãi trên GitHub.
![](https://images.viblo.asia/c088566a-740d-4306-abd5-93d0b02abc16.png)

### 2. [React Bootstrap](https://react-bootstrap.github.io/)

Bootstrap 3 components xây dựng trên React. Với hơn [13k stars](https://github.com/react-bootstrap/react-bootstrap), thư viện này vẫn phổ biến và hữu ích mặc dù vẫn đang chờ sự support cho Bootstrap 4.
![](https://images.viblo.asia/21a5bd3b-16b6-4960-8b6d-bae9ba9b2ab7.png)


### 3. [Grommet](http://grommet.io/)

React UI component đẹp, tập trung chủ yếu vào trải nghiệm cần thiết. Với hơn [3k stars](https://github.com/grommet/grommet), những thành phần thiết kế cực kỳ đơn giản nhưng design đẹp.

![](https://images.viblo.asia/e9551217-243d-407f-86db-10f206ab0b41.png)

### 4. [Ant Design React](https://ant.design/docs/react/introduce)

React components phổ biến theo hướng dẫn và thông số kỹ thuật thiết kế Ant, được viết bằng typescript và hỗ trợ luồng công việc phong phú.

![](https://images.viblo.asia/3cc5aa38-8c26-44ef-a5d1-6f9681b4c92e.png)

[Ant design components](https://github.com/ant-design/ant-design)

### 5. [Blueprint](http://blueprintjs.com/)

Một React-based UI với bộ công cụ hỗ trợ cho web với hơn 9k stars và 100 người đóng góp chính.

Components được viết bằng TS và với styled là Sass cho phát triển nhanh.

![](https://images.viblo.asia/4da78f31-879f-44ae-9b87-e79eb71f3902.png)

[Blueprint react components](https://github.com/palantir/blueprint)

### 6. [Reactstrap](https://reactstrap.github.io/)

Đây là thư viện đơn giản cho React bootstrap 4. Với hơn [4k stars](https://github.com/reactstrap/reactstrap) và gần 100 người đóng góp. Nó là lựa chọn phổ biến rộng rãi cho fan của bootstrap.

![](https://images.viblo.asia/b9099ce5-e065-4d8a-87e9-54eef34f2ae3.png)

Reactstrap [buttons](https://reactstrap.github.io/components/buttons/)

### 7. [React Toolbox](http://react-toolbox.io/#/)

Nó được xây dựng dựa trên những thứ tốt nhất như Css Modules(viết bằng SASS), Webpack và ES6. Với gần 8k stars điều này rất quan trọng.

![](https://images.viblo.asia/881cad21-3ae3-4c32-8346-3f0b29fd2727.png)

[React Toolbox components](http://react-toolbox.io/#/components)

### 8. [React Desktop](http://reactdesktop.js.org/)

Một [thư viện](https://github.com/gabrielbull/react-desktop) UI components với giao diện người dùng cho bản web, vài tính năng cho macOS Sierra và Windows 10.

![](https://images.viblo.asia/8c38768d-965a-46c1-956f-ae7c9b9a5e2d.png)

[React Desktop Window Component](http://reactdesktop.js.org/docs/mac-os/window/)


### 9. [Semantic UI React](https://react.semantic-ui.com/introduction)

Semantic UI React là một tích hợp chính thức React cho [Sematic UI](https://semantic-ui.com/). Với [6.5K stars](https://github.com/Semantic-Org/Semantic-UI-React) được sử dụng bởi Netflix, Amazon và nhiều tổ chức lớn khác.

![](https://images.viblo.asia/8252daf5-9d2b-474c-b1ff-0a138110661a.png)

[React Semantic-UI Dimmer component](https://react.semantic-ui.com/modules/dimmer)

### 10. [Onsen UI React](https://onsen.io/react/)

Với hơn 100 components được thiết kế đặc biệt cho Material và Flat, với Onsen framework và React để xây dựng các ứng dụng apps khác nhau.

![](https://images.viblo.asia/b6d586b1-fbe3-4a3e-9825-13373406303d.png)

[Onsen UI React components](https://onsen.io/theme-roller/)

### 11. [Rebass](http://jxnblk.com/rebass/)

Thư viện React UI component và thiết kế hệ thống, xây dựng với styled-components và styled-system. Với gần [4k stars](https://github.com/jxnblk/rebass) nó là một sự lựa chọn của fan Styled-components.

![](https://images.viblo.asia/1c975bb4-6919-4a36-89b6-21ce2c2d2cb7.png)

[Rebass React UI components](http://jxnblk.com/rebass/getting-started)

### 12. [Elemental UI](http://elemental-ui.com/)
Là một bộ công cụ UI cho web và ứng dụng sử dụng React.js với 4k stars và hiện tại được phát triển bởi tác giả của Keystone.js tại Thinkmill
![](https://images.viblo.asia/dad21e6b-9a5d-4840-89c6-ae64f53a2f80.png)

[Elemental UI modal component](http://elemental-ui.com/)

### 13. [Belle](http://nikgraf.github.io/belle/#/?_k=19jslm)

Là tập hợp của React components tối ưu cho mobile và desktop với các kiểu tùy chỉnh cao có thể được cấu hình ở level base dành riêng cho từng phần của chúng.

![](https://images.viblo.asia/e084b350-2e53-45e9-968b-f8c40be36de9.png)

[React belle components](http://nikgraf.github.io/belle/#/component/card?_k=2mkdb1)

### 14. [Prime React](https://www.primefaces.org/primereact/#/)
Mặc dù chỉ có hơn [400 stars](https://github.com/primefaces/primereact), chúng tôi yêu dự án này cung cấp một lựa chọn phong phú và độc đáo của hơn 60 UI components với nhiều chủ đề.

![](https://images.viblo.asia/9bf6ba87-e486-414e-98ed-b93496524ad3.png)

[Prime-React Chart Component](https://www.primefaces.org/primereact/#/polarareachart)

### 15. [Atlaskit](https://atlaskit.atlassian.com/)

Bộ [React UI](https://bitbucket.org/atlassian/atlaskit-mk-2) chính thức của Atlasssian là việc triển khai kỹ thuật [Atlassian Design Guidelines](http://atlassian.design/) (ADG). Và có từng gói riêng.

![](https://images.viblo.asia/41f9fc8d-4e61-4a91-8da6-ca06bb7f1127.png)

[Atlaskit Multi-select React component](https://atlaskit.atlassian.com/packages)

### 16. [Fabric](https://developer.microsoft.com/en-us/fabric)
A set of React components for building experiences for Office and Office 365, written in Typescript. At over 2.5K stars it’s worth checking out (quick-start).
Một tập hợp React components để xây dựng các trải nghiệm cho Office và Office 365, được viết bằng Typescript. Với hơn [2.5K stars](https://github.com/OfficeDev/office-ui-fabric-react) đáng để trả tiền ([quick-start](https://dev.office.com/fabric#/get-started)).
![](https://images.viblo.asia/e70bd099-f357-431a-9225-d23bbe7ef7ac.png)

Fabric [Nav-Bar](https://developer.microsoft.com/en-us/fabric#/components/nav) React Component

### 17. [Carbon Components](http://react.carbondesignsystem.com/)

React components được thiết kế theo [hệ thống Carbon](http://carbondesignsystem.com/) của IBM. Bạn cũng có thể dùng thử online với [CodeSandBox](https://codesandbox.io/s/x2mjypo6pp).

![](https://images.viblo.asia/e10c8e4e-a42a-42e1-b496-92aec0607120.png)


### 18. [Khan React Components](https://github.com/Khan/react-components)


Những React UI components này gần như không được chú ý, được phát triển bởi học viện Khan đã được release với gần 1k stars

![](https://images.viblo.asia/1d5e469a-9b12-4f95-afa8-2a4b7e76b490.png)

Khan’s blur-input [React component](http://khan.github.io/react-components/#tooltip)

### 19. [Gestalt by Pinterest](https://github.com/pinterest/gestalt)

Thư viện này là bộ thiết kế tỉ mỉ của React UI xây dựng theo các blocks để đồng bộ hóa và nâng cao kết nối giữa các team của Pinterest.

Gestalt React [search-filed component](https://pinterest.github.io/gestalt/#/SearchField)

### 20. [Material Components Web](https://material.io/components/web/)

Được phát triển bởi một nhóm các kỹ sư và nhà thiết kế tại Google, thư viện này thay thế react-mdl, đã đạt gần [7k sao](https://github.com/material-components/material-components-web/).

![](https://images.viblo.asia/e7b25846-e139-4206-aad6-c4b5b5b9e412.png)

[Material-web components](https://material.io/components/web/catalog/)

### 21. [React MD](https://react-md.mlaursen.com/)

Bộ công cụ tuyệt vời và được đề xuất cao để xây dựng một ứng dụng web với Thiết kế Material Design của Google với các chủ đề và style có thể tùy chỉnh rất cao (Sass).
