## 日記と化したコメント達 – Comment của Dev thiếu tự tin
```
//よくわかりませんのでこうなりました。間違ってたらごめんなさい。
```
Vì không hiểu gì cả nên tôi code thế này. Nếu sai thì xin lỗi nhìu nhé ^^

```
/* お願い…動いてくれ… */
```
Làm ơn sửa cho nó chạy dùm cái

```
// このコードは最低だ。知ってるだろ？俺もわかってんだよ。
// 俺をバカだと呼んでくれ。
```
Code này là “tối thiểu” rồi. Bạn hiểu gì ko ? tôi cũng không.    
Gọi tôi là thằng khùng cũng được nếu thích
```
// このコードが必要あるのかわかんないけど、なんか消すのがこわい
```
Đoạn code này cần hay không cũng không chắc lắm, nhưng không dám xoá.

```
// なにが何だかわからない
```
Chỗ này làm gì cũng éo hỉu

## 日記と化したコメント達 – Comment của Dev tự kỷ
## 
```
# もーーーー！JavaScript DOMなんて大嫌いだ！
```
Lại nữa ! Đúng là chúa ghét thể loại JavaScript DOM

```
// もっといい仕事見つけよっと
```
Tôi đang tìm công việc mới tốt hơn

```
// これ書いた奴バカじゃねーの…全然動かねーよ！
```
Thằng điên nào đẻ ra đoạn này vậy, hoàn toàn không chạy được

```
// あーイライラする。帰った方がよさそうだ。
```
Thiệt là bực bội quá đi mà. Có lẽ nên về nhà

```
// 昼食はチキン・ベーコンラップ
```
Thực đơn bữa trưa gồm gà vs bánh thịt hun khói

```
// 修正するにはコーヒーが必要
```
Cần phải táng cà phê vào mới fix được

```
// 課長がやればいいのに。
```
Đáng lẽ phải bắt ông sếp code cái này … haizzz

## 俺のコードが最強説 - Khi coder khẳng định mình 
```
// これは魔法。触っちゃダメ。
```
Đây là ảo thuật ! cấm sờ vào

```
// もし動作したなら、これは田中によって書かれたコードだ。
// 動作しなかったら誰が書いたかわからない。
```
Nếu đoạn này chạy được thì đích thực là code của Tanaka
Còn không chạy được thì chắc là do ai đó làm, tôi không liên quan.
```
// コメントは書かない
// 説明しづらい
// だから読みづらいはず
```
Code không comment, không giải thích thì hiểu bằng mắt (chắc là comment của ông nào hốt sh*t)
