Để làm việc được nhanh và hiệu quả hơn với eclipse thì chúng ta cần biết 1 số thủ thuật sau
1. Tự động sinh comment bằng cách vào Source -->  Renerate Element Coment hoặc ấn tổ hợp phím ALT + Shift + J
2. Để thay đổi Code template của phần commnet chúng ta vào Windown --> Preferences --> Java --> Code Style -->  Code Template 
![](https://images.viblo.asia/2f3a881a-a704-4c8d-b8e4-b35884261525.png)
3.  Nhấn tổ hợp phím CTR+SHIFT+ C để comment 1 dòng
4.  CTR + SHIFT + F để format lại code
5.  CTR + SHIFT + / để comment nhiều dòng
6.  Có thể thay đổi các phím tắt theo ý muốn của mình bằng cách vào Windown --> Preferences -->General --> Keys
![](https://images.viblo.asia/c13814b9-e218-4f07-a988-3ae7e97a36e6.png)
