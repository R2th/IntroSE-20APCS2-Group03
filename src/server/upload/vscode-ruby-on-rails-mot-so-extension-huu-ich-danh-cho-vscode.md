Visual Studio Code, được duy trì bởi Microsoft, là một trong những trình soạn thảo mã nguồn mở phổ biến nhất hiện có. Nó giúp tối ưu hóa việc viết code cho anh em coder, đồng thời có 1 kho tàng các extension để hỗ trợ. Trong bài viết này, chúng ta cùng điểm mặt qua một số extension thú vị mà mình hay sử dụng nhé.

## 1. GitLens

1 công cụ hỗ trợ mạnh mẽ cho anh em coder trong việc quản lý source code. Nó kết hợp với Git cho phép bạn xem nội dung thay đổi của từng dòng code ngay trên VSCode 1 cách trực quan. Với Gitlens, bạn có thể biết được tác giả của đoạn code, thông tin commit và thời gian thay đổi của đoạn code đó.

![Screen Shot 2022-06-15 at 14.13.50.png](https://images.viblo.asia/310ae50d-bf87-4744-b245-c518b12a4308.png)

Ngoài ra, bạn cũng có thể dễ dàng truy cập được đến commit và pull request của những sự thay đổi trong code, giúp hỗ trợ việc điều tra bug và bảo trì.

## 2. Git History

Thêm một công cụ khác khá hay ho để hỗ trợ quản lý source code cho anh em nữa đó là **Git History**. Extension này cung cấp chức năng truy vết lịch sử thay đổi của file, so sánh file giữa các lần thay đổi, từ đó giúp chúng ta dễ dàng điều tra được vấn đề liên quan đến bug, mất code hay tìm ra chủ nhân đầu tiên của đoạn code 😆

![](https://images.viblo.asia/d6d5d8ad-46e0-49ed-8ea0-bbb7377a79f0.png)

## 3. Dracula Official
Một trong những theme đang rất được ưa chuộng trong cộng đồng đó là **Dracula**. Với màu sắc thuộc gam màu tối, dịu, phù hợp cho anh em coder

![Screen Shot 2022-06-15 at 15.58.32.png](https://images.viblo.asia/2ad7deae-cd7c-4bb1-a6a6-8e01fbd854db.png)

## 4. vscode-icons
1 phần không thể thiếu khi sử dụng vscode đó là icon cho các folder và file 😂. Thử tưởng tượng 1 source code có hàng tá folder đều giống nhau, rất dễ bị nhầm lẫn và hại mắt nhỉ. Ở đây mình sử dụng **vscode-icons** vì mình thấy kho icon của nó khá đa dạng cũng như đẹp, dễ nhìn.

 ![](https://images.viblo.asia/3e612110-7480-45a1-83ba-bb8e0edce3ca.png) ![Screen Shot 2022-06-15 at 16.06.46.png](https://images.viblo.asia/7ad3a103-decd-4d60-aae6-686de1b054ff.png)
 
 ## 5. Tabnine
Extension này có tính năng gợi ý code, giúp chúng ta viết code nhanh hơn. Tabnine có kho dữ liệu được training bằng AI, tổng hợp từ nhiều cộng đồng coder, nên khả năng gợi ý code khá tốt và chính xác.

![Screen Shot 2022-06-15 at 16.12.43.png](https://images.viblo.asia/81575a5e-1e7b-4b76-b66f-8f1dcd573112.png)

## 6. ruby-rubocop
Rubocop là 1 gem khá thân thuộc và quan trọng giúp cho anh em coder Ruby on Rails cải thiện chất lượng code. Với extension này, **Rubocop** sẽ run trực tiếp và báo lỗi ngay trên code của bạn, giúp các bạn dễ follow được các vi phạm trong code của mình tối ưu hơn.

![Screen Shot 2022-06-15 at 16.15.51.png](https://images.viblo.asia/fc01835a-dd9e-4683-9bcf-26121eda6ca9.png)
