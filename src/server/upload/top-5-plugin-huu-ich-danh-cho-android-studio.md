## [2. ADBWIFI](https://plugins.jetbrains.com/plugin/7856?pr=idea)
Đây là một plugin cho phép bạn debug app của bạn trên device thông qua wi-fi mà không cần phải kết nối dây dợn rườm rà.

## [3. ADB Idea](https://github.com/pbreault/adb-idea)
Là một plugin giúp bạn tăng tốc các tác vụ thường xuyên phải sử  dụng với end-device
Các command được cung cấp bao gồm:

* Uninstall App
* Kill App
* Start App
* Restart App
* Clear App Data
* Clear App Data and Restart

![](https://images.viblo.asia/30f90b3a-a2cd-47a2-a9a6-ef4909f5ca0c.png)https://images.viblo.asia/30f90b3a-a2cd-47a2-a9a6-ef4909f5ca0c.png

Các command có thể dễ dàng được truy cập bằng cách nhấn tổ hợp phím:
* Mac OSX: Ctrl+Shift+A
* Windows/Linux: Ctrl+Alt+Shift+A

## [4. Key Promoter](https://plugins.jetbrains.com/plugin/4455?pr=clion)
Một "thầy giáo khó tính" luôn chờ trực bạn click vào toolbar hay một button nào đó để nhắc nhở bạn có thể thực hiện thao tác tương đương với các phím tắt trên bàn phím.
Rất phù hợp dành cho các bạn muốn tăng tốc lập trình.

## [5. Code Glance](https://plugins.jetbrains.com/plugin/7275?pr=clion)
Một panel show minimap code tương tự như trên Sublime giúp bạn tìm tìm kiếm code dễ dàng hơn.
