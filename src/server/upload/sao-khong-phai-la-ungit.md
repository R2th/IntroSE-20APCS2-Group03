# I. Giới thiệu
Người xưa ( và người nay) có câu: "*Rảnh rỗi sinh nông nỗi*". Mà trong mùa dịch giữa lòng thành phố Đà Nẵng như thế này, với công sức chống dịch của cả nước, mình quyết định viết bài này với mong muốn lấy đi ít rảnh rỗi của em và các anh em bằng hữu đã đọc tới đây :handshake: Thôi không lan man nữa, đi vào thằng vấn đề luôn cho nó máu.

Hiện nay trôi nổi trên thị trường là rất nhiều Git GUI Client để giúp các bạn quản lí Git như GitHub Desktop, SmartGit, SourceTree, ... Tuỳ các bạn muốn xài cái nào thì xài, mình không quản được, nhưnng mình xin giới thiệu một công cụ mà mình thấy khá tiện lợi là **Ungit** :open_mouth:.

# II. Cài đặt

Đây là link công cụ : https://www.npmjs.com/package/ungit

Cách cài đặt khá đơn giản, chỉ như cài thêm 1 thư viện mới vậy :laughing: Yêu cầu là node.js (≥ 10.18.0), npm (≥ 6.13.4, comes with node.js) và git (≥ 1.8.x) nha các bạn.

``` npm install -g ungit ```

Cơm nước xong xuôi bạn chỉ cần vào thư mục chứa source, gõ ```ungit```, tiện quá phải không nào.

![](https://images.viblo.asia/0b3702c4-6ccc-4bbc-9283-22e3ea564904.png)

Bạn chờ vài giây, nó sẽ chạy 1 server local và mở trang quản lí ở trình duyệt cho bạn.

![](https://images.viblo.asia/42ca4748-0db5-4391-bf33-376c68250472.png)
